## Security Features

- **Client-Side Encryption:** All motion data encrypted before transmission  
- **Immutable Aggregation:** Computation occurs on encrypted data, preventing tampering  
- **Confidential Analytics:** Therapists view only aggregated progress metrics  
- **Privacy by Design:** No identifiable patient data is exposed at any stage
